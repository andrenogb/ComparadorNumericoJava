Comparador de Números Inteiros em Java
Este programa recebe dois números do usuário e realiza uma análise comparativa entre eles.

Funcionalidades:

Verifica se os números são iguais.

Identifica qual número é maior que o outro em caso de diferença.

Exibe mensagens claras de comparação no console.

Loop de repetição para múltiplas consultas.

Conceitos de Programação:

Operadores Relacionais: Uso de == (igualdade) e > (maior que).

Estruturas de Decisão: Encadeamento de if, else if e else.

Controle de Fluxo: Uso do for (;;) (loop infinito) com interrupção por break baseada na escolha do usuário.
